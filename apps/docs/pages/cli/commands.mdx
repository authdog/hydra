# Commands

Hydra offers a number of commands to manage your Hydra worker.

- [version](#version) - Display Hydra installed version
- [help](#help) - Display Help for Hydra commands
- [generate-schema](#generate-schema) - Generate a GraphQL unified schema of the GraphQL endpoints referenced in hydra configuration

## version


<Tabs items={['default']}>
  <Tabs.Tab>
    ```bash copy
    hydra -V
    ```
  </Tabs.Tab>
</Tabs>


## help

<Tabs items={['default']}>
  <Tabs.Tab>
    ```bash copy
    hydra -h
    ```
  </Tabs.Tab>
</Tabs>


## generate-schema

<Tabs items={['default', 'custom config path']}>
  <Tabs.Tab>
    ```bash copy
    hydra generate-schema
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash copy
    hydra generate-schema -c config/hydra.config.ts
    ```
  </Tabs.Tab>
</Tabs>

`-c`, `--config`: Path to Hydra configuration path, default being Path to Hydra configuration path, default being `<hydra.config.ts>`
