# Commands

Hydra offers a number of commands to manage your Hydra worker.

- [version](#version) - Display Hydra installed version
- [help](#help) - Display Help for Hydra commands
- [generate-schema](#generate-schema) - Generate a GraphQL unified schemas of the GraphQL endpoints referenced in hydra configuration

## version

```bash copy
hydra -V
```

## help

```bash copy
hydra -h
```

## generate-schema

```bash copy
hydra generate-schema
```
